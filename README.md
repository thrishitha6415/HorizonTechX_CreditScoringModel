# Credit Scoring Model

## Horizon TechX Machine Learning Internship – Task 1

### 📌 Project Overview

This project is developed as part of the **Horizon TechX Machine Learning Internship**. The objective is to build a machine learning model that predicts a customer's creditworthiness based on their financial information.

The model classifies customers into different credit score categories by analyzing various financial attributes.

---

## 🎯 Objective

Develop a machine learning model to predict customer credit scores using historical financial data and evaluate its performance using standard classification metrics.

---

---

## 📂 Dataset

**Dataset:** Credit Score Classification Dataset

**Source:** https://www.kaggle.com/datasets/parisrohan/credit-score-classification

The dataset contains customer financial information such as:

- Age
- Annual Income
- Monthly In-hand Salary
- Number of Bank Accounts
- Number of Credit Cards
- Interest Rate
- Number of Loans
- Delay from Due Date
- Outstanding Debt
- Credit Utilization Ratio
- Credit History Age
- Payment Behaviour
- Credit Mix
- Credit Score (Target Variable)

---

## ⚙️ Project Workflow

1. Import required libraries.
2. Load the dataset.
3. Perform data cleaning and preprocessing.
4. Handle missing values.
5. Encode categorical variables.
6. Split the dataset into training and testing sets.
7. Train the Random Forest Classifier.
8. Make predictions on the test dataset.
9. Evaluate the model.
10. Visualize the results.
